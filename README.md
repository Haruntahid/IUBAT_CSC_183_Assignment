# Simple Banking Transaction System (C)

This project is a **Simple Banking Transaction System** developed using the **C programming language** as part of an academic assignment.

The system simulates basic banking operations such as:
- Deposit money
- Withdraw money
- Check account balance
- View transaction history (last 10 transactions)
